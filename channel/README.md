## hummer-channel js-sdk安装配置

（1）npm包

npm install hummer-channel

```javascript
import Hummer from 'humer-channel'
```

（2）采用http(CDN)方式引用聊天室js_sdk
```javascript
<script charset="utf-8" src=" https://***.**.com/hummer-channel-x.x.x.js"></script>
```
使用js-sdk的业务部署在后端服务器。
